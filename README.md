# VC 导航站
VC 导航站是一个简单的网站导航系统，收集了中文 VOCALOID 的各种常用网站。

此项目是 Tianyi Network Web 的一部分。

## 构建
此项目使用 GitHub Actions 自动构建，构建文件请参见 [deploy.yml](https://github.com/luotianyi-dev/web-vcportal/blob/main/.github/workflows/deploy.yml)。
